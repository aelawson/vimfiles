
========

This is my Neovim editor setup.
Feel free to fork it
and submit a pull request if you found any bug.

Instalation
-----------

    $ git clone https://github.com/kris89/vimfiles.git ~/vimfiles
    $ cd ~/vimfiles
    $ chmod +x ./install_neovim.sh
    $ ./install_neovim.sh
    $ nvim
    $ :PlugInstall
    $ :UpdateRemotePlugins
