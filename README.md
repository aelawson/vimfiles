
========

This is my Vim editor setup
Feel free to fork it
and submit a pull request if you found any bug.

Instalation
-----------

    $ cd ~
    $ git clone https://github.com/kris89/vimfiles.git
    $ cd vimfiles
    $ git submodule init
    $ git submodule update
    $ cd ~
    $ ln -s vimfiles/vim .vim
    $ ln -s vimfiles/vimrc .vimrc

----------------

* NERDTree
* CtrlP
* Filesearch

* Surround
* Snipmate
* Auto pairs
* Multiple cursors
* Airline
* Supertab
* Fugitive
* Emmet
* Syntastic
* Easymotion

* Blade syntax
* Json syntax
* Twig syntax

* Javascript syntax
