
========

This is my Vim editor setup.
Feel free to fork it
and submit a pull request if you found any bug.

Instalation
-----------

    $ git clone https://github.com/kris89/vimfiles.git ~/vimfiles
    $ cd ~/vimfiles
    $ chmod +x ./install.sh
    $ ./install.sh

Run Vim and accept Bundles(Plugins) Installation.
